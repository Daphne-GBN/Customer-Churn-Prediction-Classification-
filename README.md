# Customer Churn Prediction (Machine Learning)

## Overview
Customer churn is a major challenge for telecom companies. Predicting which customers are likely to leave helps companies take preventive actions such as offering discounts, better plans, or loyalty benefits.

This project builds a machine learning classification model to predict whether a telecom customer will churn (leave the service) or stay.

The model is trained on the Telco Customer Churn dataset and evaluates multiple machine learning algorithms.

---

## Dataset

Dataset Source:
https://www.kaggle.com/datasets/blastchar/telco-customer-churn

The dataset contains telecom customer information such as:

• Demographics  
• Services subscribed  
• Account information  
• Billing details  

Target Variable:

Churn  
Yes → Customer leaves  
No → Customer stays

---

## Key Features Used

Important features used for prediction include:

• Contract type  
• Tenure  
• Monthly charges  
• Internet service  
• Payment method  
• Total charges  

---

## Feature Engineering

Two additional features were created to improve model performance.

Customer Lifetime Value (CLV)

CLV = MonthlyCharges × Tenure

This represents the estimated revenue generated by a customer.

Monthly Charge Category

Customers were grouped into categories based on their monthly bill:

Low  
Medium  
High

---

## Machine Learning Models Used

Three classification models were trained and evaluated:

Logistic Regression  
Random Forest  
CatBoost

---

## Evaluation Metrics

The models were evaluated using the following metrics:

F1 Score  
ROC-AUC Score

These metrics help measure how well the model identifies customers who are likely to churn.
